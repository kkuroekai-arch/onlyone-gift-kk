# お問い合わせフォーム / GAS デプロイ手順

このサイトのお問い合わせフォームは、Google Apps Script (GAS) を使ってメールを送信します。GAS は無料で、Googleアカウントがあれば誰でも使えます。

---

## ① GASプロジェクトを作成

1. Googleアカウントにログインした状態で [https://script.google.com/](https://script.google.com/) を開く
2. 左上の **「新しいプロジェクト」** をクリック
3. 左上の「無題のプロジェクト」をクリックして、**「K&k Contact Form」** などに名前変更

## ② コードを貼り付け

1. デフォルトで開いている `Code.gs` の中身を全部削除
2. このフォルダの `gas-code.gs` の中身をすべてコピー&ペースト
3. **`CONFIG`** ブロックの中身を確認:
   ```js
   TO_EMAIL: 'info@nishihama-iron.jp',  // ← 受信用メアド
   FROM_NAME: 'K&k Co., Ltd.',
   COMPANY_URL: 'https://onlyone-gift-kk.com', // ← 公開URLに変更
   ```
4. 上部のフロッピーディスクアイコンで **保存** (`Ctrl+S` / `⌘+S`)

## ③ 権限を承認

1. 上部メニューバーの「実行」ボタンの左にある関数選択で **`testSend`** を選ぶ
2. **「実行」** をクリック
3. 「承認が必要です」というダイアログが出る → **「権限を確認」**
4. 自分のGoogleアカウントを選択
5. 「Google で確認されていません」という警告が出たら、**「詳細」→「(プロジェクト名)に移動」**
6. **「許可」** をクリック
7. 実行が完了すると、`TO_EMAIL` に指定したメアド宛にテストメールが届きます ✅

## ④ Webアプリとしてデプロイ

1. 右上の **「デプロイ」** → **「新しいデプロイ」**
2. 左上の歯車アイコン → **「ウェブアプリ」** を選択
3. 設定:
   - **説明**: `K&k Contact Form v1` (任意)
   - **次のユーザーとして実行**: `自分(your-email@gmail.com)`
   - **アクセスできるユーザー**: ⚠️ **`全員`** (重要・ここを間違えると動きません)
4. **「デプロイ」** をクリック
5. 「ウェブアプリURL」が表示される (例: `https://script.google.com/macros/s/AKfycb.../exec`)
6. このURLを **コピー** しておく

## ⑤ config.js にURLを貼り付け

⚠️ URLを貼るのは **`config.js`** です(`index.html` ではありません)。
こうすることで、今後 `index.html` を更新しても、URL設定が消えなくなります。

`config.js` を開いて、次の行を探します:

```js
window.GAS_URL = "YOUR_GAS_WEB_APP_URL_HERE";
```

`YOUR_GAS_WEB_APP_URL_HERE` の部分を、④でコピーしたURLに置き換えます:

```js
window.GAS_URL = "https://script.google.com/macros/s/AKfycb..../exec";
```

ダブルクォート `"` は消さないように注意してください。

保存して、ブラウザでサイトを開いて、お問い合わせフォームから送信テストをしてください。
- 自分のメアドで送信 → 自動返信が届くか確認 ✅
- `info@nishihama-iron.jp` (管理者宛て) にも通知メールが届くか確認 ✅

> 💡 **重要:** 今後 `index.html` を新しいものに差し替えるときは、
> `config.js` は **アップロードし直さない** でください。
> `config.js` をそのまま残しておけば、URL設定はずっと保持されます。

## トラブルシューティング

### 「メールが届かない」
- `TO_EMAIL` の値が正しいか確認
- GASエディタで `testSend` 関数を実行してエラーが出ていないか確認
- スパムフォルダにも入っていないか確認

### 「送信時にエラーになる」
- デプロイ設定で `アクセスできるユーザー: 全員` になっているか再確認
- URLが正しくコピペできているか(末尾は `/exec` で終わる)
- ブラウザのコンソール(F12 → Console)でエラーメッセージを確認

### 「コードを修正したい」
- GASエディタでコードを修正 → 保存
- **「デプロイ」 → 「デプロイを管理」** から既存のデプロイの右上の鉛筆アイコンをクリック
- **「バージョン」を「新しいバージョン」に変更** → デプロイ
- ⚠️ URLは変わりません(同じURLのまま新しいコードが反映される)

### 「過去のお問い合わせ履歴を残したい」 (任意)
1. 新しいGoogleスプレッドシートを作成
2. URLから「`/d/`」と「`/edit`」の間の長い文字列をコピー
   - 例: `https://docs.google.com/spreadsheets/d/1abc...xyz/edit` → `1abc...xyz` の部分
3. `gas-code.gs` の下のほうにある `SHEET_ID` を書き換え:
   ```js
   const SHEET_ID = '1abc...xyz';
   ```
4. 保存 → 新バージョンとしてデプロイ
5. 以降のお問い合わせが、すべてシートにも記録されます

---

## セキュリティと無料枠

- **無料**: GASは個人アカウントで1日100通までメール送信可能
- **スパム対策**: フォームに見えない「ハニーポット」フィールドがあり、bot送信は自動的にブロックされます
- **個人情報**: メール送信のみで、外部サービスにはデータが送られません
